# 06. 버튼 컴포넌트

버튼 컴포넌트는 AbstractButton을 상속받은 하위 클래스들을 말합니다. 버튼 컴포넌트에는 JButton,

JToggleButton, JRadioButton, JCheckBox가 있는데, 모두 사용자가 마우스로 클릭하여 사용

할 수 있도록 되어 있습니다. 다음은 버튼 컴포넌트의 상속 관계를 보여 준다.

AbstractButton

JButton

JToggleButton

JCheckBox

JRadioButton

버튼 컴포넌트를 마우스로 클릭하면 모두 ActionEvent가 발생합니다. 그래서 addActionListener ( )

메소드로 ActionListener 객체를 등록하여 이벤트를 처리할 수 있습니다.

JButton

JButton은 이미지와 텍스트로 구성된 일반적인 버튼을 만들 때 사용합니다. JButton의 setText ( )

메소드는 버튼의 텍스트를 설정하고, setIcon ( ) 메소드는 버튼의 이미지를 설정합니다.

JButton jButton  =  new JButton();
jButton.setText("새문서");
jButton.setIcon( new ImageIcon( getClass().getResource("new.gif") );

다음 예제는 텍스트, 이미지, 텍스트+이미지 버튼을 생성합니다. 버튼의 이벤트 처리는 파일 열기 대

화상자를 보여주도록 했다. 이 대화상자를 활용하는 방법은 13절에서 설명합니다.

JToggleButton

JToggleButton은 선택된 상태와 그렇지 않은 두 가지 상태를 가지는 버튼입니다. 생성 방법은

JButton과 유사해서 텍스트와 이미지를 설정할 수 있습니다.

JToggleButton jToggleButton  =  new JToggleButton();
jToggleButton.setText("확인");
jToggleButton.setIcon(new ImageIcon( getClass().getResource("ok.gif") );

JToggleButton은 선택된 상태를 가지는 버튼이기 때문에 ActionListener보다는 다음과 같

이 ItemListener로 이벤트를 처리하는 것이 좋다. ItemEvent의 getStateChange ( ) 메소드는

JToggleButton이 선택되었을 경우 ItemEvent.SELECTED 상수값을 리턴합니다.

jToggleButton.addItemListener(new ItemListener() {
```java
@Override
public void itemStateChanged(ItemEvent e) {
if(e.getStateChange()  = =  ItemEvent.SELECTED) {
//선택된 상태
} else {
//해제된 상태

}

}

});


```

JToggleButton은 단독으로도 사용 가능하지만, JToggleButton 두 개를 ButtonGroup에 포함

시키면 두 버튼을 배타적으로 선택할 수 있습니다. ButtonGroup 내부에서는 하나의 버튼만이 선택된

상태로 존재할 수 있기 때문입니다.

ButtonGroup buttonGroup  =  new ButtonGroup();
buttonGroup.add( jToggleButton1 );
buttonGroup.add( jToggleButton2 );

다음 예제는 [On] [Off] 토글 버튼과, 배타적으로 선택할 수 있는 [Start]와 [Stop] 버튼을 생성합니다.

그리고 [Start]와 [Stop] 버튼을 클릭하면 메시지 다이얼로그를 보여 준다. 다이얼로그에 대한 내용

은 13절에서 자세히 설명합니다.

JRadioButton

JRadioButton은 JToggleButton의 하위 클래스로, 둥근 모양의 선택과 텍스트를 함께 보여주는

버튼입니다. JRadioButton은 동일한 ButtonGroup에 포함되어 한 번에 하나의 JRadioButton만

선택된 상태를 가진다. 다음 코드는 남자와 여자 둘 중 하나를 선택할 수 있습니다.

JRadioButton jRadioButton1  =  new JRadioButton();
jRadioButton1.setText("남자");

JRadioButton jRadioButton2  =  new JRadioButton();
jRadioButton2.setText("여자");

ButtonGroup buttonGroup  =  new ButtonGroup();
buttonGroup.add(jRadioButton1);
buttonGroup.add(jRadioButton2);



JRadioButton은 마우스로 클릭했을 때 ActionEvent가 발생하므로 ActionListener로 이벤트를

처리할 수 있습니다. 다음 예제는 두 개의 JRadioButton 중에 하나를 선택하면 JLabel의 이미지가 변

경되도록 하였다.

JCheckBox

JCheckBox는 사각형의 체크박스와 텍스트를 함께 보여주는 컴포넌트입니다. 이 컴포넌트도

JToggleButton을 상속하며 체크와 언체크의 두 가지 상태를 갖는다. JRadioButton과 차이점은

개별적으로 선택할 수 있다는 것입니다. 다음 코드는 축구와 농구를 개별 선택할 수 있도록 합니다.



JCheckBox jCheckBox1  =  new JCheckBox();
jCheckBox1.setText("축구");

JCheckBox jCheckBox2  =  new JCheckBox();
jCheckBox2.setText("농구");

JCheckBox도 마우스로 클릭했을 때 ActionEvent가 발생하므로 ActionListener로 이벤트를 처

리할 수 있습니다. JCheckBox가 선택되었는지 확인하는 방법은 isSelected ( ) 메소드의 리턴값을 확

인하면 됩니다. true가 리턴되면 선택이 된 것입니다.

다음 예제는 JCheckBox 체크 상태에 따라 JLabel의 이미지를 변경합니다.

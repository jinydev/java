# Chapter 06. 조건문

## 학습목표

1.  **흐름 제어**: 위에서 아래로 흐르는 프로그램의 순서를 바꾸는 방법을 이해합니다.
2.  **조건문**: 상황에 따라 다른 코드를 실행하는 `if`문과 `switch`문을 배웁니다.

## 목차

### [6.1 코드 실행 흐름 제어](./flow-control)
물이 위에서 아래로 흐르듯 실행되는 코드의 흐름을 댐(제어문)으로 조절하는 원리를 배웁니다.

"만약 ~라면"이라는 조건에 따라 갈림길을 만드는 방법을 배웁니다.

### [6.3 switch 문](./switch)
자판기 버튼처럼 정해진 값에 따라 바로바로 실행할 코드를 선택하는 방법을 배웁니다.

### [6.4 (심화) 개선된 switch 문](./switch-pattern)

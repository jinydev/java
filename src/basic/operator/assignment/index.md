# 5.6 대입 연산자

## 1. 기본 대입 (`=`)

오른쪽 값을 왼쪽에 저장합니다.

```java
int score = 80;
```

## 2. 복합 대입 (`+=`, `-=`, ...)

계산과 저장을 한 번에 하는 **단축키**입니다.

|  연산자  | 풀이        | 설명          |
| :------: | :---------- | :------------ |
| **`+=`** | `x = x + y` | 더하고 저장   |
| **`-=`** | `x = x - y` | 빼고 저장     |
| **`*=`** | `x = x * y` | 곱하고 저장   |
| **`/=`** | `x = x / y` | 나누고 저장   |
| **`%=`** | `x = x % y` | 나머지를 저장 |

```java
int money = 1000;
money += 500; // money는 1500이 됨
```

# 5.5 논리 연산자

여러 가지 조건을 조합할 때 사용합니다.

|  연산자  | 이름             | 설명                     |
| :------: | :--------------- | :----------------------- |
| **`&&`** | **AND** (그리고) | 둘 다 참이어야 참        |
|   **`    |                  | `**                      | **OR** (또는) | 둘 중 하나라도 참이면 참 |
| **`!`**  | **NOT** (반대)   | 참이면 거짓, 거짓이면 참 |
| **`^`**  | **XOR**          | 둘이 서로 다르면 참      |

## 예시

```java
int age = 20;
boolean hasTicket = true;

// 성인이고(AND) 티켓이 있어야 입장 가능
if (age >= 19 && hasTicket) {
    System.out.println("입장 가능");
}
```

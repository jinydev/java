# 5.4 비교 연산자

두 값을 비교해서 **참(`true`)**인지 **거짓(`false`)**인지 판단합니다.

|  연산자  | 의미        | 설명                         |
| :------: | :---------- | :--------------------------- |
| **`==`** | 같다        | 두 값이 같으면 참            |
| **`!=`** | 다르다      | 두 값이 다르면 참            |
| **`>`**  | 크다        | 왼쪽이 더 크면 참            |
| **`>=`** | 크거나 같다 | 왼쪽이 더 크거나 같으면 참   |
| **`<`**  | 작다        | 오른쪽이 더 크면 참          |
| **`<=`** | 작거나 같다 | 오른쪽이 더 크거나 같으면 참 |

## 주의사항: 문자열 비교

문자열(`String`)이 같은지 비교할 때는 **`==`를 쓰면 안 됩니다!**
반드시 **`.equals()`** 기능을 써야 합니다.

```java
String s1 = "Java";
String s2 = "Java";

// s1 == s2 (위험! 가끔 틀릴 수 있음)
boolean result = s1.equals(s2); // (안전!)
```

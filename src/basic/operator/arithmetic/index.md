# 5.2 산술 연산자

## 1. 사칙연산과 나머지 ➗

자바의 기본 산술 연산자 5가지입니다.

| 연산자  | 설명        | 예시            |
| :-----: | :---------- | :-------------- |
| **`+`** | 더하기      | `1 + 1` -> `2`  |
| **`-`** | 빼기        | `2 - 1` -> `1`  |
| **`*`** | 곱하기      | `2 * 3` -> `6`  |
| **`/`** | 나누기 (몫) | `10 / 3` -> `3` |
| **`%`** | **나머지**  | `10 % 3` -> `1` |

> **주의**: 정수끼리 나누면 **몫만** 나옵니다. (`5 / 2 = 2`)
> 소수점까지 보고 싶으면 하나라도 실수(`5.0 / 2`)여야 합니다.

## 2. 문자열 더하기

`+` 연산자는 숫자뿐만 아니라 **글자도 이어 붙여줍니다.**

```java
String str = "JDK" + 21; // "JDK21"
```

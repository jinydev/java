# Chapter 03. 자바 시작하기

## 학습목표

1.  **실행 과정**: 소스 코드를 작성하고 실행하는 흐름을 익힙니다.
2.  **코드 용어**: 패키지, 클래스, 메인 메소드 등의 기본 용어를 이해합니다.
3.  **주석과 실행문**: 코드를 설명하는 주석과 마침표 역할을 하는 세미콜론을 배웁니다.

## 목차

### [3.1 소스 작성부터 실행까지](./execution-process)
`Hello.java` 파일을 만들고, 컴파일러(`javac`)와 실행기(`java`)를 통해 "Hello, Java"를 출력해 봅니다.

### [3.2 코드 용어 이해](./code-terminology)
자바 코드의 기본 뼈대인 패키지(폴더), 클래스(파일), `main()` 메소드(시작 버튼)의 역할을 알아봅니다.

### [3.3 코드 주석 달기](./comments)
코드에 설명을 적어놓는 '메모(주석)' 기능을 배웁니다. 컴퓨터는 이 부분을 무시합니다.

### [3.4 실행문과 세미콜론](./statements)
자바의 문장은 반드시 세미콜론(`;`)으로 끝나야 한다는 규칙을 배웁니다.

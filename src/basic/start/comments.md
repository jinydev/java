# 3.3 코드 주석 달기

## 1. 개발자를 위한 비밀 메모 📝

코드를 짜다 보면 "이게 무슨 기능이지?" 하고 헷갈릴 때가 있습니다.
이때 **주석(Comment)**을 사용해서 설명을 적어두면 좋습니다.

컴퓨터(컴파일러)는 주석을 **완벽하게 무시**합니다.
따라서 주석을 아무리 많이 써도 프로그램 성능에는 전혀 영향이 없습니다.

## 2. 주석의 종류

### 1) 한 줄 주석 (`//`)
`//` 뒤에 적은 내용은 그 줄 끝까지 주석이 됩니다.

```java
// 이것은 주석입니다. 컴퓨터는 읽지 못합니다.
System.out.println("Hello"); // 코드 뒤에 붙여도 됩니다.
```

### 2) 여러 줄 주석 (`/* ... */`)
`/*`로 시작해서 `*/`로 끝날 때까지 모든 내용이 주석이 됩니다.

```java
/*
  작성자: 홍길동
  작성일: 2024.02.20
  내용: 자바 첫 실습
*/
public class Hello { ... }
```

> **팁**: 좋은 개발자는 코드를 어렵게 짜고 주석을 많이 다는 사람이 아니라, **주석 없이도 읽기 쉽게 코드를 짜는 사람**입니다. 그래도 복잡한 로직엔 주석이 필수입니다!

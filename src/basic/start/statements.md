# 3.4 실행문과 세미콜론

## 1. 마침표를 찍어주세요 (`;`)

우리말이나 영어에서 문장이 끝나면 마침표(`.`)를 찍듯이,
자바에서는 명령(실행문)이 끝나면 반드시 **세미콜론(`;`)**을 찍어야 합니다.

```java
int x = 1   // (X) 에러 발생! 마침표가 없어요.
int y = 2;  // (O) 성공!
```

## 2. 실행문의 종류

*   **변수 선언**: `int x;` (그릇 만들기)
*   **값 저장**: `x = 10;` (그릇에 담기)
*   **메소드 호출**: `System.out.println("Hello");` (일 시키기)

세미콜론을 빠뜨리는 것은 초보자가 가장 많이 하는 실수 중 하나입니다.
**"명령 끝!"**이라는 의미로 `;`를 잊지 마세요.

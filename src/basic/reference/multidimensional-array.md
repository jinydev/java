# 8.7 다차원 배열

## 1. 아파트 짓기 🏢

1차원 배열이 **기차(한 줄)**라면,
2차원 배열은 **아파트(여러 층)**와 같습니다.
행(층)과 열(호수)이 있습니다.

```java
int[][] scores = new int[2][3]; // 2층짜리 아파트, 각 층에 3호수씩
```

## 2. 값 읽고 쓰기

```java
scores[0][1] = 90; // 0층 1호에 90 저장
// [층][호수]
```

## 3. 계단식 배열

각 층마다 방의 개수가 다를 수도 있습니다.
(1층은 2개, 2층은 3개 ...)
메모리를 효율적으로 쓸 수 있습니다.

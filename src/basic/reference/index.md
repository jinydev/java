# Chapter 08. 참조 타입

## 학습목표

1.  **참조(Reference)**: 데이터가 저장된 '주소'를 가리키는 개념을 이해합니다.
2.  **배열(Array)**: 데이터를 기차처럼 줄줄이 엮어서 관리하는 방법을 배웁니다.
3.  **열거(Enum)**: 정해진 값(요일, 계절 등)만 사용할 수 있는 한정된 타입을 배웁니다.

## 목차

### [8.1 데이터 타입 분류](./classification)
값 자체를 저장하는 '기본 타입'과 주소를 저장하는 '참조 타입'의 차이를 알아봅니다.

### [8.2 메모리 사용 영역](./memory-area)
자바 프로그램이 사용하는 메모리 공간(메소드, 스택, 힙)을 알아봅니다.

### [8.3 참조 변수의 비교](./equality)
`==` 연산자가 기본 타입일 때와 참조 타입일 때 어떻게 다르게 동작하는지 배웁니다.

### [8.4 null과 NullPointerException](./null)
"참조할 대상이 없다"는 의미인 `null`과, 이로 인해 발생하는 대표적인 에러를 배웁니다.

### [8.5 문자열(String) 타입](./string)
문자열도 사실은 참조 타입이라는 것을 이해하고, 문자열을 다루는 방법을 배웁니다.

### [8.6 배열(Array) 타입](./array)
같은 타입의 데이터를 여러 개 묶어서 저장하는 배열을 배웁니다.

### [8.7 다차원 배열](./multidimensional-array)
아파트처럼 행과 열로 이루어진 2차원 이상의 배열을 배웁니다.

### [8.8 배열 복사](./array-copy)
배열의 크기를 늘리거나 내용을 복사하는 방법을 배웁니다.

### [8.9 향상된 for 문](./enhanced-for)
배열의 모든 항목을 간편하게 꺼낼 수 있는 반복문을 배웁니다.

### [8.10 열거(Enum) 타입](./enum)
요일이나 계절처럼 정해진 몇 가지 값만 가질 수 있는 타입을 배웁니다.

# Chapter 08. 참조 타입 (Reference Types)

### 학습 목표 (Learning Objectives) 🎯

이 챕터를 마치면 여러분은 다음과 같은 능력을 갖추게 됩니다:
1. **메모리 구조 이해**: 스택(Stack)과 힙(Heap) 메모리의 차이를 알고 데이터를 어떻게 분류하는지 설명할 수 있습니다.
2. **참조의 개념 마스터**: 변수가 실제 값이 아닌 '메모리 주소'를 가리킨다는 참조 타입의 핵심 원리를 완벽히 이해합니다.
3. **배열 뽀개기**: 자바에서 가장 많이 쓰이는 자료구조인 배열(Array)과 다차원 배열을 자유자재로 만들고 다룰 수 있습니다.
4. **문자열 능숙도 향상**: `String` 객체가 메모리에서 어떻게 동작하는지 파악하고 효율적으로 활용합니다.
5. **에러 대처 능력**: 'NullPointerException'의 원인을 파악하고 스스로 해결할 수 있습니다.

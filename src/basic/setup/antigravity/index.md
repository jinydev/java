## 주요 기능

## 코딩 영단어 학습 📝

코딩에서 영어 단어의 의미만 정확히 이해해도 절반은 성공입니다! 오늘 배운 핵심 영단어들을 다시 한번 짚고 넘어가 볼까요?

*   **`Antigravity`**: 반중력, 안티그라비티. (코드 작성의 수고로움을 덜어주는 AI 코딩 어시스턴트의 이름)
*   **`Assistant`**: 어시스턴트, 조수. (개발을 돕는 보조 프로그램)
*   **`Pair Programming`**: 페어 프로그래밍, 짝 프로그래밍. (두 명이 짝을 지어 하나의 모니터를 보며 함께 코딩하는 개발 방법론)

# 6. 인텔리제이 설치 및 소개

IntelliJ IDEA는 JetBrains사에서 개발한 자바 통합 개발 환경(IDE)입니다. 강력한 코드 완성, 리팩토링 기능, 그리고 다양한 플러그인을 제공하여 많은 개발자들에게 사랑받고 있습니다.

## 1. 인텔리제이 에디션

인텔리제이는 두 가지 에디션으로 제공됩니다.

*   **Community Edition**: 무료 오픈 소스 버전으로, 자바 개발을 위한 대부분의 기능을 제공합니다. 학습 목적이나 개인 프로젝트에는 충분합니다.
*   **Ultimate Edition**: 유료 버전으로, 웹 개발 및 엔터프라이즈 프레임워크 지원 등 고급 기능을 제공합니다. 학생이나 교사는 무료 라이선스를 신청할 수 있습니다.

이 책에서는 무료인 **Community Edition**을 기준으로 설명합니다.

## 2. 인텔리제이 설치

1.  **다운로드**: [JetBrains 공식 웹사이트](https://www.jetbrains.com/ko-kr/idea/download/)에 접속하여 운영체제에 맞는 설치 파일을 다운로드합니다.
2.  **설치**: 다운로드한 파일을 실행하여 설치를 진행합니다. 설치 과정에서 특별한 설정 없이 기본값을 유지해도 됩니다.
3.  **실행**: 설치가 완료되면 인텔리제이를 실행합니다. 초기 설정 화면에서 테마 등을 선택하고 [Next] 버튼을 클릭합니다.

## 3. 새 프로젝트 생성

1.  인텔리제이 시작 화면에서 **New Project**를 클릭합니다.
2.  **Name**에 프로젝트 이름을 입력합니다. (예: `HelloJava`)
3.  **Location**에서 프로젝트가 저장될 경로를 지정합니다.
4.  **Language**는 `Java`, **Build system**은 `IntelliJ`를 선택합니다.
5.  **JDK**에서 설치된 JDK 버전을 선택합니다. 만약 JDK가 목록에 없다면 [Add JDK...]를 클릭하여 JDK 설치 경로를 지정합니다.
6.  [Create] 버튼을 클릭하여 프로젝트를 생성합니다.

이제 인텔리제이에서 자바 프로그래밍을 시작할 준비가 되었습니다.

# Chapter 02. 개발 환경 구축

## 학습목표

1.  **JDK 설치**: 자바 개발에 필요한 도구 상자(JDK)를 설치합니다.
2.  **환경 변수**: 컴퓨터 어디서든 자바를 쓸 수 있게 설정합니다.
3.  **IDE 설치**: 개발을 편하게 도와주는 도구(이클립스, 인텔리제이, VS Code)를 설치합니다.

## 목차

### [2.1 JDK 설치](./jdk-installation)
자바의 핵심 도구인 JDK(Java Development Kit)를 윈도우와 맥에 설치하는 방법을 알아봅니다.

### [2.2 환경 변수 설정](./environment-variable)
운영체제에게 "자바 여기 있어!"라고 알려주는 환경 변수 설정법을 배웁니다.

### [2.3 이클립스 설치](./eclipse-installation)
전통적인 자바 개발 도구인 이클립스(Eclipse) 설치 방법을 알아봅니다.

### [2.4 인텔리제이 설치 및 소개](./intellij-installation)
현업에서 가장 많이 쓰는 인텔리제이(IntelliJ IDEA) 설치 방법을 알아봅니다.

### [2.5 VS Code 설치](./vscode-installation)
가볍고 강력한 에디터인 Visual Studio Code 설치 방법을 알아봅니다.

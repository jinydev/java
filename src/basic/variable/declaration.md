# 4.1 변수 선언

## 1. 변수란? (데이터를 담는 상자) 📦

**변수(Variable)**는 값을 저장할 수 있는 메모리 공간입니다.
쉽게 말해, **"데이터를 담는 이름표 붙은 상자"**라고 생각하면 됩니다.

*   하나의 변수에는 **하나의 값**만 저장할 수 있습니다.
*   변수에는 다양한 타입(정수, 실수, 문자 등)의 값을 저장할 수 있습니다.

## 2. 변수 선언 (상자 준비하기)

변수를 쓰려면 먼저 "어떤 데이터를 담을 상자인지"와 "상자의 이름"을 정해야 합니다.

```java
int age;       // 정수(int)를 담을 상자, 이름은 age
double height; // 실수(double)를 담을 상자, 이름은 height
```

## 3. 값 저장 (상자에 물건 넣기)

대입 연산자(`=`)를 사용합니다. 수학에서는 "같다"는 뜻이지만, 프로그래밍에서는 **"오른쪽 값을 왼쪽 상자에 넣어라"**는 뜻입니다.

```java
age = 25;      // age 상자에 25를 넣음
height = 175.5 // height 상자에 175.5를 넣음
```

## 4. 초기화 (Initialization)

변수를 선언함과 동시에 값을 넣을 수도 있습니다. 이를 **초기화**라고 합니다.

```java
int score = 90; // 상자를 만들면서 바로 90을 넣음
```

> **주의**: 값이 들어있지 않은(초기화되지 않은) 변수는 사용할 수 없습니다!
> 빈 상자에서 물건을 꺼낼 수 없는 것과 같습니다.

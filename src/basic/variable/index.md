# Chapter 04. 변수와 타입

## 학습목표

1.  **변수**: 데이터를 저장하는 그릇(변수)의 개념을 이해합니다.
2.  **데이터 타입**: 정수, 실수, 문자, 논리 등 다양한 데이터의 종류를 배웁니다.
3.  **타입 변환**: 데이터의 종류를 바꾸는 방법(자동, 강제)을 배웁니다.

## 목차

### [4.1 변수 선언](./declaration)
데이터를 저장할 그릇(변수)을 만들고, 이름을 붙이는 방법을 배웁니다.

### [4.2 정수 타입](./integer-type)
숫자(정수)를 저장하는 `byte`, `short`, `int`, `long` 타입을 배웁니다.

### [4.3 문자 타입](./char-type)
글자 하나를 저장하는 `char` 타입을 배웁니다.

### [4.4 실수 타입](./float-type)
소수점이 있는 숫자를 저장하는 `float`, `double` 타입을 배웁니다.

### [4.5 논리 타입](./boolean-type)
참/거짓(`true`/`false`)을 저장하는 `boolean` 타입을 배웁니다.

### [4.6 문자열 타입](./string-type)
여러 글자(문장)를 저장하는 `String` 타입을 배웁니다.

### [4.7 자동 타입 변환](./promotion)
작은 그릇의 물을 큰 그릇으로 옮길 때 자동으로 일어나는 변환을 배웁니다.

### [4.8 강제 타입 변환](./casting)
큰 그릇의 물을 작은 그릇으로 옮길 때 넘치지 않게 주의하며 옮기는 방법을 배웁니다.

### [4.9 변수 사용 범위](./scope)
변수가 어디까지 살아있을 수 있는지(생존 범위)를 배웁니다.

### [4.10 콘솔 출력](./console-output)
화면에 데이터를 출력하는 `System.out.println` 등을 배웁니다.

### [4.11 키보드 입력](./keyboard-input)
키보드로 데이터를 입력받는 `Scanner` 사용법을 배웁니다.

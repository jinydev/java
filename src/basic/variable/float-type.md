# 4.4 실수 타입

실수(Floating Point)는 소수점이 있는 숫자(3.14, -0.5)입니다.

| 타입         | 크기   | 정밀도   | 설명                                      |
| :----------- | :----- | :------- | :---------------------------------------- |
| `float`      | 4 byte | 낮음     | 소수점 7자리 정도 정확                    |
| **`double`** | 8 byte | **높음** | **가장 많이 씀**. 소수점 15자리 정도 정확 |

## 1. `double` (기본 실수)
`int`가 정수의 기본이듯, `double`이 실수의 기본입니다.
(Double이라는 이름은 `float`보다 2배 더 정밀하다는 뜻입니다.)

```java
double pi = 3.14159;
```

## 2. `float`
값 뒤에 **`f`** 또는 **`F`**를 붙여야 합니다.

```java
float weight = 65.5f;
```

## 3. 정밀도 차이
`double`이 훨씬 더 정교한 계산을 할 수 있습니다.

```java
float f = 0.1234567890123456789f;
double d = 0.1234567890123456789;

System.out.println(f); // 0.12345679 (뒤가 잘림)
System.out.println(d); // 0.12345678901234568 (훨씬 정확)
```

> **핵심**: 소수점이 나오면 무조건 **`double`**을 쓰세요.

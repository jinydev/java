# 1.1 프로그래밍 언어와 자바

## 1. 사람과 컴퓨터의 대화 🗣️

컴퓨터는 0과 1밖에 모르는 기계입니다. 마치 **외계인**과 같습니다.
우리가 "화면을 켜줘!"라고 말해도 컴퓨터는 알아듣지 못합니다. 컴퓨터는 `10110100...` 같은 **기계어(Machine Language)**만 이해하기 때문입니다.

## 2. 통역사 (Programming Language)

사람의 말(한국어, 영어)을 컴퓨터의 말(기계어)로 바꿔주는 **통역사**가 필요합니다.
이것이 바로 **프로그래밍 언어(Programming Language)**입니다.

*   **사람**: "화면을 켜줘" (소스 코드 작성)
*   **프로그래밍 언어(자바)**: (번역 중...)
*   **컴퓨터**: `010101...` (실행!)

## 3. 고급 언어 vs 저급 언어

### 1) 저급 언어 (Low-Level)
*   기계어에 가깝습니다. (예: 어셈블리어)
*   컴퓨터가 바로 이해해서 빠르지만, 사람이 배우기 너무 어렵습니다.
*   마치 외계어로 직접 대화하려는 것과 같습니다.

### 2) 고급 언어 (High-Level)
*   사람의 언어(영어)에 가깝습니다. (예: **Java**, C, Python)
*   "If (만약)", "While (동안)" 처럼 읽기 쉽습니다.
*   **컴파일러(Compiler)**라는 번역기가 기계어로 바꿔줍니다.
*   우리가 배울 **자바(Java)**는 대표적인 고급 언어입니다.

# 1.2 자바의 역사

## 1. 제임스 고스링과 그린 프로젝트 🌿

1991년, 썬 마이크로시스템즈의 **제임스 고스링(James Gosling)** 박사는 가전제품(TV, 셋톱박스 등)에 들어갈 소프트웨어를 만들고 있었습니다.

![재임스 고스링](./img/gosling.png)

하지만 가전제품마다 들어가는 칩(CPU)이 다 달라서, 일일이 프로그램을 다시 짜야 하는 불편함이 있었습니다.

> **"어디서든 다 돌아가는 언어는 없을까?"**

그래서 만든 것이 바로 **'Oak(오크)'**라는 언어였고, 나중에 이름이 **'Java(자바)'**로 바뀝니다. (자바 커피를 좋아해서 지었다는 설이 있습니다 ☕)

![그림](./img/oak_java.png)

## 2. 인터넷의 등장과 성공 🚀

처음엔 가전제품용이었지만, 때마침 1995년 **인터넷(Web)**이 폭발적으로 성장했습니다.
"운영체제에 상관없이 실행된다"는 자바의 특징이 인터넷 환경(전 세계 다양한 컴퓨터 연결)과 딱 맞아떨어졌습니다.
그래서 자바는 웹 프로그래밍의 표준 언어로 자리 잡게 되었습니다.

## 3. 자바의 철학

> **"Write Once, Run Anywhere"**
> (한 번 작성하면, 어디서든 실행된다)

이것이 자바를 세계 1위 언어로 만든 핵심 철학입니다.
개발자가 `Windows`에서 개발해서 배포하면, 사용자가 `Mac`을 쓰든 `Linux`를 쓰든 똑같이 잘 돌아갑니다.

# 1.3 자바의 특징

자바가 20년 넘게 사랑받는 이유는 강력한 장점들 때문입니다.

## 1. 어디서든 실행된다 (이식성) 🌏

> **"Write Once, Run Anywhere" (한 번 작성하면, 어디서든 실행된다)**

일반적인 프로그램은 윈도우용, 맥용을 따로 만들어야 합니다.
하지만 자바는 **JVM(자바 가상 머신)**이라는 만능 어댑터가 있어서, 윈도우든 맥이든 리눅스든 **똑같은 코드로 실행**됩니다.
마치 전 세계 어디를 가도 꽂을 수 있는 **여행용 멀티 어댑터**와 같습니다.

## 2. 객체지향 언어 (조립식) 🧱

자바는 **레고 블록(객체)**을 조립해서 성(프로그램)을 만드는 방식입니다.
*   **부품화**: 타이어, 엔진, 핸들을 따로 만들어서 자동차를 조립합니다.
*   **재사용**: 고장 나면 타이어만 갈아 끼우면 됩니다.
*   이런 방식을 **객체지향 프로그래밍(OOP)**이라고 합니다.

## 3. 메모리 자동 관리 (청소 로봇) 🧹

C언어 같은 옛날 언어는 개발자가 메모리를 직접 쓰고 지워야 했습니다. (까먹으면 컴퓨터가 멈춤!)
자바는 **가비지 컬렉터(Garbage Collector)**라는 청소 로봇이 내장되어 있어서, 쓰지 않는 메모리를 알아서 싹 치워줍니다.
개발자는 메모리 걱정 없이 기능 구현에만 집중하면 됩니다.

## 4. 풍부한 무료 라이브러리 📚

전 세계 수많은 개발자가 만들어둔 **오픈소스 라이브러리**가 엄청나게 많습니다.
지도 기능, 그래프 기능, PDF 만들기 등 필요한 기능은 대부분 이미 만들어져 있습니다. 우리는 가져다 쓰기만 하면 됩니다.

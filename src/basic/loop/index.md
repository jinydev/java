# Chapter 07. 반복문

## 학습목표

1.  **반복문**: 코드를 여러 번 실행하게 만드는 `for`, `while` 문을 배웁니다.
2.  **제어**: 반복을 중간에 멈추거나(`break`), 이번만 건너뛰는(`continue`) 방법을 배웁니다.

## 목차

### [7.1 for 문](./for)
"10번 반복해!"처럼 횟수가 정해진 반복 작업을 처리하는 방법을 배웁니다.

### [7.2 while 문](./while)
"비가 그칠 때까지 반복해!"처럼 조건에 따라 반복하는 방법을 배웁니다.

### [7.3 do-while 문](./do-while)
일단 한 번은 실행하고 나서, 계속할지 말지 결정하는 방법을 배웁니다.

### [7.4 break 문](./break)
반복문을 즉시 탈출하는(멈추는) 방법을 배웁니다.

### [7.5 continue 문](./continue)
이번 회차만 건너뛰고 다음 반복으로 넘어가는 방법을 배웁니다.

# 자바 언어 소개

## 학습목표

프로그래밍 언어의 역할과 자바의 특징을 이해하고, 자바 프로그램이 실행되는 원리인 바이트코드와 자바 가상 머신(JVM)의 역할에 대해 학습합니다.

## 목차

### [1. 프로그래밍 언어와 자바](./01)
사람이 이해하는 언어와 컴퓨터가 이해하는 기계어의 차이, 그리고 이를 연결해주는 프로그래밍 언어와 컴파일의 개념에 대해 알아봅니다.

### [2. 자바의 역사와 제임스 고스링](./02)
자바를 탄생시킨 제임스 고스링과 그린 프로젝트, 그리고 "Write Once, Run Anywhere"라는 철학이 어떻게 자바를 성공으로 이끌었는지 알아봅니다.

### [3. 자바의 특징](./03)
자바가 가진 강력한 특징들(이식성, 객체지향, 메모리 자동 관리 등)이 개발자에게 어떤 이점을 주는지 상세히 살펴봅니다.

### [4. 바이트코드 파일과 자바 가상 머신](./04)
자바 소스 파일이 컴파일되어 생성되는 바이트코드 파일(.class)과 이를 실행시키는 자바 가상 머신(JVM)의 역할에 대해 알아봅니다.

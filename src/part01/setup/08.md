# 8. 안티그라비티 소개

**안티그라비티(Antigravity)**는 Google Deepmind 팀이 개발한 강력한 AI 코딩 어시스턴트입니다. 개발자와의 페어 프로그래밍을 통해 코드 작성, 디버깅, 리팩토링 등 다양한 작업을 수행할 수 있습니다.

## 주요 기능

*   **코드 작성 및 수정**: 사용자의 요구사항에 맞춰 코드를 작성하거나 기존 코드를 수정합니다.
*   **파일 및 프로젝트 관리**: 여러 파일을 생성, 수정하고 프로젝트 구조를 관리할 수 있습니다.
*   **터미널 명령 실행**: 필요한 패키지 설치, 빌드, 테스트 실행 등 터미널 명령을 수행합니다.
*   **웹 검색 및 리소스 조회**: 최신 기술 문서나 해결 방법을 검색하여 제공합니다.
*   **시각화 및 디자인**: UI 목업 생성 및 데이터 시각화 기능을 제공합니다.

## 활용 방법

안티그라비티는 단순한 코드 완성을 넘어, 복잡한 개발 작업을 함께 해결하는 동반자입니다. 학습 과정에서 모르는 개념을 질문하거나, 작성한 코드의 리뷰를 요청하고, 새로운 기능을 구현하는 데 활용해 보세요.

안티그라비티와 함께라면 더 빠르고 효율적으로 프로그래밍을 학습하고 개발할 수 있습니다.

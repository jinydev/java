# 개발 환경 구축

## 학습목표

자바 프로그래밍을 위한 JDK 설치, 환경 변수 설정, 그리고 통합 개발 환경(IDE) 설치 방법을 배웁니다. 이클립스(Eclipse)와 비주얼 스튜디오 코드(VS Code) 중 선호하는 도구를 선택하여 설치할 수 있습니다.

## 목차

### [1. JDK 설치](./01)
자바 프로그램을 개발하고 실행하기 위해 필요한 Java SE(Standard Edition)의 구현체인 JDK(Java Development Kit)를 운영체제별로 설치하는 방법을 알아봅니다.

### [2. 환경 변수 설정](./02)
JDK 설치 후, 운영체제 어디에서든 자바 명령어를 사용할 수 있도록 `JAVA_HOME` 및 `Path` 환경 변수를 설정하는 방법을 알아봅니다.

### [3. 이클립스 설치](./03)
전통적으로 자바 개발에 가장 많이 사용되는 강력한 통합 개발 환경(IDE)인 이클립스(Eclipse)를 설치하는 방법을 알아봅니다.

### [4. 이클립스 프로젝트 생성](./04)
이클립스에서 자바 소스 코드를 작성하고 관리하기 위한 단위인 '자바 프로젝트'를 생성하는 방법을 알아봅니다.

### [5. 이클립스 소스 작성부터 실행](./05)
이클립스에서 간단한 자바 소스 파일(Hello.java)을 작성하고, 컴파일 및 실행하여 결과를 확인하는 과정을 실습합니다.

### [6. 인텔리제이 설치 및 소개](./06)
강력한 기능으로 많은 개발자에게 사랑받는 JetBrains의 IntelliJ IDEA 설치 및 프로젝트 생성 방법을 알아봅니다.

### [7. VS Code 설치](./07)
최근 많이 사용되는 가볍고 강력한 에디터인 Visual Studio Code를 설치하고, 자바 개발을 위한 'Coding Pack for Java' 및 확장 팩을 설정하는 방법을 알아봅니다.

### [8. 안티그라비티 소개](./08)
AI 코딩 어시스턴트인 안티그라비티를 소개하고, 개발 학습 및 실무에서의 활용 방법을 알아봅니다.

# 20.3 Client Tool 설치

SQL Developer는 Oracle DB 개발 및 관리를 간편하게 해주는 무료 Client Tool이다.

1.  오라클 공식 웹사이트에서 SQL Developer를 다운로드한다.
2.  `C:\Oracle\sqldeveloper` 경로에 압축을 해제한다.
3.  `sqldeveloper.exe`를 실행한다.
4.  새 데이터베이스 접속을 생성한다.
    *   **Name**: thisisjava
    *   **사용자 이름**: java
    *   **비밀번호**: oracle
    *   **호스트 이름**: localhost
    *   **포트**: 1521
    *   **SID**: orcl

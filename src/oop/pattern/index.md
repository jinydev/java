## 코딩 영단어 학습 📝

코딩에서 영어 단어의 의미만 정확히 이해해도 절반은 성공입니다! 오늘 배운 핵심 영단어들을 다시 한번 짚고 넘어가 볼까요?

*   **`Design`**: 디자인, 설계. (건물을 짓기 전에 도면을 그리듯, 코드를 짜기 전에 전체적인 구조와 청사진을 그리는 프로그래밍 설계 작업)
*   **`Pattern`**: 패턴, 양식. (수많은 선배 개발자들이 코딩을 하다가 "어? 이런 문제는 이렇게 푸니까 항상 좋더라!" 하고 정립해 둔 모범 답안이자 공식)

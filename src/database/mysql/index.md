# Chapter 20. JDBC (MySQL)

본 부록은 MySQL을 사용하여 JDBC 프로그래밍을 학습하는 분들을 위한 가이드입니다. 
Oracle을 사용하는 분들은 [데이터베이스 입출력](../jdbc/index.md)을 참고하세요.

## 학습목표

*   MySQL을 사용하여 JDBC 프로그래밍을 할 수 있다.
*   MySQL Workbench를 사용하여 MySQL을 관리할 수 있다.
*   MySQL Connector/J를 사용하여 MySQL에 연결할 수 있다.
*   MySQL Connector/J를 사용하여 MySQL에서 데이터를 저장, 수정, 삭제, 조회할 수 있다.
*   MySQL Connector/J를 사용하여 MySQL에서 트랜잭션을 처리할 수 있다.
*   MySQL Connector/J를 사용하여 MySQL에서 게시판을 구현할 수 있다.
